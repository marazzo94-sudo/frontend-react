# frontend-react

React front-end for a crypto dashboard starter project.

## Development

```bash
npm install
npm run dev
```

Source files are located in the `src/` directory with the following structure:

```
src/
  components/
    common/
    layout/
    charts/
    tables/
  pages/
  hooks/
  context/
  assets/
    images/
    icons/
    fonts/
  utils/
```
Import any shared UI elements from `src/components`.
